Business Purpose

Skills Developed
Problem Solving, Innovation
Materials / Setup
Eggs, tape, straws, paper, rubber bands, cotton balls, balloons, newspapers, drop area.
How to Play
  1. Introduce the activity goal and connect it to problem solving, innovation.
  2. Split participants into the recommended format: 3-6 participants per team.
  3. Give the group the core challenge: design protective device for egg drop.
  4. Run the activity within the time limit while the facilitator observes teamwork, decision quality, and participation.
  5. Debrief using reflection questions and capture workplace applications or commitments.
Review Questions
  • What did the activity reveal about problem solving in the team?
  • Where did the team lose time, clarity, or alignment?
  • How can the same behavior be applied to real workplace projects?
Key Takeaway
Egg Drop Challenge helps employees practice problem solving, innovation through a structured workplace activity. The value comes from the debrief: teams should connect what happened in the game to how they plan, communicate, decide, and improve at work.
